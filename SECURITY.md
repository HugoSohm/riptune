# Security Policy

## Supported Versions

RipTune is currently in its early stages. We recommend using the latest version of the application.

| Version | Supported          |
| ------- | ------------------ |
| >= 0.1.0| :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in RipTune, please DO NOT report it as a public GitHub issue. Instead, report it privately by emailing us at: **security@riptune.hugosohm.fr**.

Please include:
1. A description of the vulnerability.
2. Steps to reproduce the issue.
3. Your suggested fix (if any).

We will verify and address the issue as soon as possible.
